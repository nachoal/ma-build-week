# Redacted Gate 0 evidence

Only review-safe derivatives belong here. Before adding an artifact, remove:

- raw learner audio and transcripts not required for the public claim;
- client secrets, authorization headers, install tokens, and standard API keys;
- stable device, session, response, item, network, and account identifiers;
- filesystem paths or metadata that identify the learner.

Permitted artifacts are bounded aggregate tables, configuration facts needed to
scope the claim, synthetic clock-alignment tests, redacted event excerpts, and
waveform figures that contain no recoverable private speech. Every artifact
must be traceable to a private source by a random local marker recorded in the
verdict without exposing the source itself.
